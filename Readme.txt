# 📊 Excel Sales Dashboard

## 🚀 Project Overview
This project is an interactive **Sales Dashboard** built in Microsoft Excel to analyze agent performance and business metrics.

It transforms raw data into meaningful insights using charts, KPIs, and dynamic filtering.

---

## 📌 Key Features
✔️ KPI Cards (Total Calls, Reach, Deals Closed, Revenue)  
✔️ Monthly Performance Trends  
✔️ Sales vs Target Comparison  
✔️ Call Drop Rate & Average Call Duration  
✔️ Agent-wise Performance Analysis  
✔️ Interactive Dashboard using Slicers  

---

## 🛠️ Tools & Skills Used
- Microsoft Excel  
- Data Cleaning  
- Pivot Tables  
- Data Visualization  
- Slicers (Dynamic Filtering)  

---

## 📊 Dashboard Insights
- Easily track overall sales performance  
- Identify top-performing agents  
- Analyze trends over time  
- Monitor key business metrics
